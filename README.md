# naruto_handsign
using mediapipe to extract hand landmark; to classify into 13 different pose for handsign(jutsu)
\n it include 12 norm and "zero" for reset sequence

## How to use
1. download release file
2. run " run.bat "
### optional
if streamlit feels laggy
`./run.bat --opencv`
